# Reflection on this project
* Always take into account the limitations of the API, such as request frequency and the type of data the API can access.
* Modifying an existing API or developing your own one can be a huge project, so it's advisable not to undertake it alone.
* If I undertake a similar project in the future, I will opt for an API that offers greater flexibility. This way, there will be more room for enhancements and customization.
